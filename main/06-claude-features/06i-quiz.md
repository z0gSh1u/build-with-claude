# 06i - 小测验

> 文件 API 用于什么？

- 扫描文件以检测病毒和恶意软件
- 压缩大文件以降低 API 成本
- 自动在不同格式之间转换文件
- **提前上传文件并在稍后引用它们，而不是直接在消息中编码**

> 您正在使用相同的大系统提示进行许多请求。有什么功能可以让您的请求更快更便宜？

- PDF 处理
- 引用
- 扩展思考
- **提示词缓存**

> Claude 中引用的主要目的是什么？

- **为了从 Claude 的回答中清晰地追踪到源文档的特定部分**
- 为了压缩大文档以加快处理速度
- 为了计算文档中的单词数量
- 为自动生成学术论文的脚注

> 当 Claude 使用扩展思维时，响应中包含哪两个部分？

- **推理过程和最终答案**
- 问题和解决方案
- 输入和输出
- 问题与答案

> 您想让 Claude 分析一个 PDF 文档。与发送图片相比，主要区别是什么？

- **将类型改为“document”并将 media_type 设置为“application/pdf”**
- 处理 PDF 文件的成本更高
- 您只能发送文本，不能在 PDF 中发送图片
- PDFs 需要特殊权限

> Claude 的代码执行工具的关键局限性是什么？

- 它只能运行 JavaScript 代码
- **它没有网络访问，并在隔离的 Docker 容器中运行**
- 它要求用户提供自己的执行环境
- 它只能处理文本文件

> 您想缓存系统提示。缓存要正常工作，最低要求是什么？

- 您必须至少发起 5 次请求
- 您必须使用扩展思维
- 内容必须在 500 个 token 以内
- **内容长度必须至少为 1024 个标记**
