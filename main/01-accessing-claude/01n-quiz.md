# 01n - 小测验

> 您想向 Claude 的 API 发送请求。您必须包含的最少信息是什么？

- 只有 API 密钥和您的问题
- 只需您的消息文本
- ☑️ API 密钥、模型名称、消息和最大 Token 数
- 您的姓名、电子邮件和消息

> 你问 Claude "披萨是什么？"它回答了。然后你问"什么配料很受欢迎？"但 Claude 不明白你在指什么。问题出在哪里？

- 你的网络连接很慢
- Claude 出故障了
- 你问得太快了
- ☑️ Claude 不记得之前的消息

> 当 Claude 处理你的文本时，它首先做什么？

- 立即生成响应
- 检查内容是否合适
- ☑️ 将其分割成更小的称为 Token 的块
- 翻译成另一种语言

> 用户抱怨您的聊天应用感觉缓慢，因为他们盯着加载动画等了 20 秒，然后所有生成的文本一次性出现。如何解决这个问题？

- 提出更短的问题
- 使用更快的互联网连接
- ☑️ 启用流式响应
- 使用不同的网络浏览器

> 你正在开发一个与 Claude 交互的网页应用。你应该在哪里存储你的 API 密钥？

- 在用户安装的手机应用中
- 在用户电脑上的文本文件中
- ☑️ 在用户无法访问的服务器上
- 在用户下载的 JavaScript 代码中

> 你正在开发一个数学辅导机器人。你希望 Claude 提供提示而不是直接答案。你应该使用什么？

- 设置非常低的字数限制
- 在消息中使用全大写字母
- ☑️ 一个解释导师角色的系统提示
- 要求用户更加具体

> 你希望 Claude 在事实问答应用中给出非常可预测、一致的答案。你应该使用什么温度设置？

- 温度对事实无关紧要
- ☑️ 低温度（接近 0.0）
- 中等温度（约 0.5）
- 高温（接近 1.0）

> 你正在开发一个需要从 Claude 获取干净 JSON 的应用，不包含任何额外文本或格式。如何只获取原始 JSON？

- 多次发送请求并选择最好的一个
- 非常礼貌地要求 Claude 只返回 JSON
- ☑️ 组合预填充消息和停止序列
- 使用非常高的温度设置
