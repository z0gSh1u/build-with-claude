# Claude 模型比较

| 模型              | 描述                         | 成本 | 相对延迟 | 支持推理 | 最佳用途                                     |
| ----------------- | ---------------------------- | ---- | -------- | -------- | -------------------------------------------- |
| **Claude Opus**   | 最高水平的智能               | 高   | 中等     | 是       | 高级软件开发、大规模架构设计、策略规划       |
| **Claude Sonnet** | 平衡质量、速度和成本         | 中   | 快       | 是       | 常见编码任务、文档创建、数据分析、流程自动化 |
| **Claude Haiku**  | 最具成本效益和延迟优化的模型 | 低   | 最快     | 否       | 快速代码补全、内容审核、数据提取、语言翻译   |
