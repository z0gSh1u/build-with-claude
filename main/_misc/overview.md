# 课程概述

这门全面的课程将教开发者如何使用 Anthropic API 将 Claude AI 集成到应用程序中。课程内容涵盖基础 API 操作、高级提示词技巧、工具集成以及构建 AI 驱动系统的架构模式。通过实践练习和实例，开发者将学习如何实现对话式 AI、检索增强生成（RAG）、自动化工作流，并利用 Claude 的多模态功能处理文本、图像和文档。

## 你将学到什么

- 配置 Anthropic API，包括 API 密钥管理和请求配置
- 实现单轮和多轮对话，包括正确的消息格式和上下文处理
- 配置系统提示词，并使用温度、流式响应和结构化输出控制模型行为
- 设计和执行提示词评估，包括测试数据集生成和自动评分系统
- 提示词工程技术，包括 XML 标签结构化、示例和清晰的指令
- 集成 Claude 的 Tool Use 能力，包括自定义工具、批量操作和网页搜索
- 构建检索增强生成（RAG）系统，包括文本分块、嵌入、BM25 搜索和上下文检索
- 使用 Claude 的扩展功能，包括扩展思考、图像分析、PDF 处理和引用
- 实施提示词缓存策略，以优化 API 使用成本并减少延迟
- 开发模型上下文协议（MCP）服务器和客户端，以实现标准化的工具和资源集成
- 部署 Anthropic 应用，包括用于自动化开发的 Claude Code 和用于 UI 自动化的 Computer Use
- 使用并行式、链式和路由式 Workflow，以及构建 Agent 系统

## 先决条件

- 熟练掌握 Python 编程
- 具备处理 JSON 数据的基本知识

## 适合谁

- 构建 AI 驱动 API 和服务的后端开发者
- 将对话式 AI 集成到 Web 应用的全栈工程师
- 进行文档处理、设计知识检索系统的数据工程师
- 借助 AI 来自动化工作的 DevOps 从业人士
- 设计可扩展的 AI 集成系统的技术架构师
- 转向 AI/ML 应用开发的软件工程师
- 聊天机器人、虚拟助手或内容生成工具的开发者
