# 04b - 项目概述

## 一个例子

我们将用如下项目来演示 Tool Use 技术：

```
教 Claude 如何为未来日期设置提醒。
例如，告诉 Claude “为我的就诊预约设置提醒，时间是从周四开始一周后”。
```

这看起来可能很简单，但实际上具备一定的挑战性：

- Claude 的时间感知有限，可能知道当前日期，但不知道确切时间
- Claude 不一定能很好地处理时间加法，尤其是向前看未来许多天的情况
- Claude 本身不具备设置提醒的功能

而工具，正是我们弥补 Claude 本身能做的事情与我们的需求之间的 Gap 的方式。

## 三个工具

为了完成上述任务，我们需要创建三个独立的工具：

- 获取当前日期和时间工具
- 日期和时间加法工具
- 设置提醒工具

我们将逐一实现这些工具，最终 Claude 将能够结合这些工具来处理自然语言请求。

Tool Use 表明了一个与 AI 合作的关键原则：当模型存在局限性时，通过工具来扩展其能力，而不是试图用提示词绕过这些局限性。
