# 08a - Anthropic 的应用介绍

在本章，我们将介绍 Anthropic 开发的两款强大应用：Claude Code 和 Computer Use。它们不仅是实用的生产力工具，更是 AI Agent 运行的完美范例。了解它们的工作原理有助于为后续构建自己的 Agent 打下基础。

## Claude Code

Claude Code 是一个基于终端的编程助手，可以帮助你处理各种编程任务。把它想象成 Claude 直接出现在你的命令行中，随时准备：编辑文件和修复错误，回答编程问题，协助开发工作流程。

## Computer Use

计算机使用将 Claude 的功能提升到了更高的水平。它是一套工具的集合，允许 Claude 与完整的桌面计算机环境进行交互。这意味着 Claude 可以：

- 访问网站和浏览互联网
- 与桌面应用程序交互
- 执行需要视觉界面导航的任务

这比纯文本交互大大扩展了可能实现的内容。

## 与 Agent 的联系

Claude Code 和 Computer Use 都是理解智能体的优秀案例研究。它们展示了使智能体有效的关键原则：

工具集成和使用

多步骤任务执行

环境交互

自主问题解决
