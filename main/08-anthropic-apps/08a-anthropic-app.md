# 08a - Anthropic 的应用介绍

在本章，我们将介绍 Anthropic 开发的两款强大应用：Claude Code 和 Computer Use。它们不仅是实用的生产力工具，更是 AI Agent 运行的完美范例。了解它们的工作原理有助于为后续构建自己的 Agent 打下基础。

## Claude Code

Claude Code 是一个基于终端的编程助手，可以帮助你处理各种编程任务。把它想象成 Claude 直接出现在你的命令行中，随时准备：编辑文件和修复错误，回答编程问题，协助开发工作流程。

## Computer Use

Computer Use 将 Claude 的功能提升到了更高的水平。它是一套工具的集合，允许 Claude 与 PC 进行交互。这意味着 Claude 可以：

- 访问网站和浏览互联网
- 与桌面应用程序交互
- 执行需要 GUI 的任务

相比纯文本交互，这大大扩展了可能实现的功能。

## 与 Agent 的联系

Claude Code 和 Computer Use 都是用于理解 Agent 的优秀案例，它们展示了使 Agent 有效的关键原则：

- 集成和使用工具
- 多步骤的任务执行
- 与环境交互
- 自主解决问题

这些优秀案例将为我们的 Agent 开发工作提供指导。
