# 08b - Claude Code 配置

Claude Code 提供了一套全面的工具来帮助进行开发工作：

- 文件操作：在项目中搜索、读取和编辑文件
- 访问终端：直接运行对话中的命令
- 访问网络：搜索文档、获取代码示例等
- MCP 支持：通过连接 MCP 服务器添加更多工具，意味着可以通过添加数据库、API 或其他专业工具来扩展功能

配置 Claude Code 只需三个步骤：

- 确保 Node.js 安装
- 使用 `npm install -g @anthropic-ai/claude-code` 全局安装 Claude Code
- 在终端运行 `claude` 并登录

设置完成后，终端中将运行着 Claude，随时准备帮助你处理任何编程任务。
