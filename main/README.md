# 用 Claude 构建应用

本项目对 Anthropic Academy 上的 Building with the Claude API 课程进行了整理，将关键内容翻译为中文呈现，补充了译者的一些体会，并在逐步添加 Jupyter Notebook 示例代码，方便读者理解和实践。

![main](https://github.com/z0gSh1u/build-with-claude/blob/master/main/README.assets/main.png?raw=true)

本课程原始链接为 [Building with the Claude API](https://anthropic.skilljar.com/claude-with-the-anthropic-api)。如果[本项目](https://github.com/z0gSh1u/build-with-claude)对你有帮助，欢迎给我一个 Star ⭐️！

本项目仅供学习交流使用，禁止用于商业用途。
